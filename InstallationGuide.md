# 🧩 Smörgåsbord Mod List – Installation Guide (7 Days to Die)

## 📌 Overview

Welcome to the Smörgåsbord Mod List Installation Guide for **7 Days to Die**.

Smörgåsbord is a Wabbajack Mod List that uses ModOrganizer 2 (MO2) to deliver one of the most modular and customizable ways to play 7DtD.

📺 Here is the video guide if you'd like to follow along: 

## ⚙️ System Requirements

✔️ If you can run vanilla → you can run Smörgåsbord


⚠️ Some mods are heavier and may require stronger hardware

**Recommended:**
- 16+GB of System Memory (RAM)
- SSD strongly recommended.
- 2000 Series Nvidia GPU or newer.
- Better than ancient CPU.

### 💡 **Lower-end system?** Disable heavier mods in MO2 and refer to the Performance guide.

## 📦 Step 2 – Install Wabbajack

👉 https://www.wabbajack.org

Install on same SSD as mod list (recommended)

Open → ⚙️ Settings → Login to NexusMods

## 📁 Step 3 – Create Folder Structure

Recommended structure:

D:\Wabbajack_ModLists\7DtD_Smorgasbord

Steps:

Create folder: Wabbajack_ModLists

Inside it, create: Smorgasbord

Copy the full path (Ctrl+A → Ctrl+C)

## ⬇️ Step 4 – Install Smörgåsbord

Inside Wabbajack:


Browse Lists


✅ Enable Non-Featured


Select **7 Days to Die**


Select **Smörgåsbord**


⚠️ Match game version in Steam


Click _Download & Install_


📌 When prompted:


Paste your folder path (Ctrl+V)

###⏳ Download Expectations

💎 **Premium**

- One click

- Fully automated

- Moves as fast as your network connection and hardware allows.


🆓 _Free_

- Manual download of every mod

- Expect hours of clicking

- Capped download speed

## 🛠️ Troubleshooting Downloads

### 🔴 Mod Failed to Download

Wait 5–10 minutes

Click Download & Install again

OR:

Manually download mod

Place in:

Wabbajack_ModLists\7DtD_Smorgasbord\downloads

### ❌ Game File Source Missing Error

Example:

Error: Data_Config_Localization.txt missing

✔️ Cause: Game version mismatch

✔️ Fix: Match Steam version to Wabbajack list version

✅ Installation Complete

When finished:

Wabbajack shows you a splash-screen of Smorgasbord saying the Mod List Installation process has completed.

You can now close Wabbajack

## ▶️ Step 5 – Launch MO2

Open your Smorgasbord folder

Run ModOrganizer2.exe

First Launch

Accept NXM Association → YES

Launch Game

Click ▶️ Play in MO2

###⚠️ ALWAYS launch Smorgasbord via MO2

💡 Pin MO2 to taskbar for convenience

## ⚙️ Mod Management Tips

Cursor-hover over mod notes in MO2 for useful info

**Read ⚠️ highlighted notes carefully**

Enable/disable mods per preference

_💡 Don’t like a mod? Disable it and move on 😄_

### ⚠️ Best Practices

❗ Only change mods on new saves

❗ Never toggle mods while game is running

❗ Some mods are NOT save-safe

### 🧪 Testing Recommendation

✔️ Start a test world before committing to a long playthrough

Example:

Try NPCCore → decide early if you love or hate it.

Not save-safe → requires new game

## 🎯 In-Game Settings Recommendations

- 🎯 Difficulty: Survivalist

- 🧟 Increase zombie speed (optional)

- 📦 Loot Abundance: ≤ 75%

## ➕ Adding Mods (Advanced Users)

### ⚠️ WARNING: Adding mods voids support

### How to Add Mods

Connect Nexus in MO2

Download mod via Nexus

OR:

Drag manual download into MO2 Downloads

Install into:

"Where you may choose to add compatible Mods"

🔒 Protect Custom Mods

Rename mods in MO2 with this prefix:

[NoDelete]

✔️ Prevents removal during updates

⚠️ Notes

Some mods may be broken or outdated

Fix manually by mimicking working mod structure if needed
